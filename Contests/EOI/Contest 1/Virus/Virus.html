Virus

Tamer is an IT manager of a large company, a new virus has hit the company, but since Tamer is a good IT manager he was able to remove it from his network, the problem is the virus renames files on an infected machine according to a special pattern. Tamer called you to write a program that will generate a script that would rename the files to their original names.

The virus renames files according to the following rules

·         Files with extension ‘.exe’ change to the reverse of the name and change the extension to ‘.inv’.

·         Files with extension ‘.dll’ circular shift left the name and change the extension to ‘.shf’.

Also it is known that there were no files with the extension ‘.inv’ nor ‘.shf’.

 

Input

Input contains file full path.

Output

Print a script to rename the infected files using the format given in the sample output. Or print correct if the file is not infected.

 

 

Sample Input / Output

Input

 

c:\programs\games\enim.inv

 

c:\programs\games\topscore.dat

 

c:\programs\common\isplayd.shf

 

Output

ren c:\programs\games\enim.inv c:\programs\games\mine.exe

 

correct

 

ren c:\programs\common\isplayd.shf c:\programs\common\display.dll
